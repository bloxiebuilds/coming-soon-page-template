# 🚧 coming-soon-page-template

> A bold, minimal coming soon page with a live countdown timer and email signup. Edit one config block and launch.

## ✨ Features

- ⏱ **Live countdown timer** — ticks down to your launch date in real time
- 📧 **Email signup** — link to any Google Form, Mailchimp, or signup URL
- 🌗 **Dark, Light, or Toggle mode**
- 🎨 **Custom accent color** — any hex code
- 🔗 **Social links** — add as many as you need
- 📱 **Fully responsive**
- ⚡ **Zero dependencies**

## 🚀 Quick Start

1. Clone or download the repo
2. Open `index.html` and edit the `CONFIG` block at the top
3. Deploy to GitHub Pages and share your link!
